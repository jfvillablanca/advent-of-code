## To run specific code (e.g. solution for 202X-XX)

### Typescript

```console
cd typescript
ts-node src/202X-XX.ts
```

### Rust

```console
cd rust
cargo run --bin 202X-XX
```

### Python

```console
cd python
python src/202X-XX.py
```
